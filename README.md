
## 📖 项目介绍

**dlna-cast** 是一款 [AstrBot](https://github.com/Soulter/AstrBot) 的插件，支持从 WebDAV/SMB 等网络存储直接推送视频到 DLNA 兼容的电视和机顶盒设备，实现便捷的远程播放控制。

> 🔥 **已支持**：哔哩哔哩 TV 版（云视听小电视）
> 🚧 **开发中**：腾讯视频、爱奇艺、芒果 TV 等主流机顶盒平台

------

## ✨ 功能特性

- 📺 **DLNA 投屏** - 自动发现局域网内的 DLNA 设备
- 📂 **多协议支持** - 支持 WebDAV、SMB 等网络存储协议
- 🎮 **播放控制** - 支持播放、暂停、停止、进度调节、音量控制
- 🎯 **平台适配** - 针对主流 TV 平台进行深度适配和优化
- 🔧 **易于扩展** - 模块化架构，方便接入新的设备平台

------

## 📦 安装方法

### 通过 AstrBot 插件市场安装

1. 打开 AstrBot 管理面板
2. 进入"插件市场"
3. 搜索 `dlna-cast` 并安装

------

## 🚀 使用方法

### 基础命令

| 命令                       | 说明                 | 示例                                                         |
| :------------------------- | :------------------- | :----------------------------------------------------------- |
| `/dlna scan`               | 扫描 DLNA 设备       | `/dlna scan`                                                 |
| `/dlna play <设备> <名称>` | 推送视频到指定设备   | `/dlna play 客厅电视 凡人第105集`<br />`/dlna play 凡人第105集` |
| `/dlna pause`              | 暂停播放             | `/dlna pause`                                                |
| `/dlna resume`             | 恢复播放             | `/dlna resume`                                               |
| `/dlna stop`               | 停止播放             | `/dlna stop`                                                 |
| `/dlna seek <秒>`          | 跳转到指定位置       | `/dlna seek 120`                                             |
| `/dlna volume <0-100>`     | 调节音量             | `/dlna volume 50`                                            |
| `/dlna xxx`                | 非上方指令，自然对话 | `/dlna 停止播放`                                             |
